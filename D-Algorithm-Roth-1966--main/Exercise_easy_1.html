D*-Algorithm - Roth(1966) ~ Easy ( 1 | 2 )

Guide:

To solve this algorithm, set the Stuck-At-Fault entries and follow the given algorithm. For the circuit showing below, the algorithm finds the stuck-at-1 (SA1), that is being generated. This Stuck-at-Fault has to be detected at the output. If you cannot solve the circuit, repeat the rules of the algorithm.

After the circuit is solved, you can view the D-Path with the "Dpath" button


For help, you can use the "hint" button 5 times otherwise reset the circuit with the "reset" button. The algorithm has an anti-puzzle code.

You can't guess any in- and outputs or unexplained in- and outputs.

Follow the rules of the algorithm and use the "solve" button if you want to compare your solution for this circuit..
